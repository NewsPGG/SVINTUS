# Отчёт по изменениям архитектуры проекта

### 1) Удаление не использующихся методов applyTurn, getCurrentPlayer из класса GameState, в MainArchitecture класс был разделён на interface и data class

### 2) Добавлен класс ActionCard, являющийся наследником абстрактного класса Card, создающий карты предписаний

### 3) Добавлен класс DefaultTurnValidator для реализации приватных функций, реализующихся в интерфейсе TurnValidator

### 4) Создание класса GameStateLogic, реализующего интерфейс IGameStateLogic. Позволяет отделить хранение состояния игры от правил её обработки.
